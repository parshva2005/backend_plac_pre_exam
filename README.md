Ticket Management System (TMS)

This is a backend project for a Ticket Management System. It is built using .NET API.

Requirements:

.NET 8 SDK
Visual Studio 2022
Setup Process:

Download the code First, download or clone the project to your computer.
If you use Visual Studio 2022:

Open the TMS.sln file.
Then Click the Green Play Button (or press F5) at the top.
